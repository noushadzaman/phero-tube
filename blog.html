What do you mean by REST API?

REST API stands for Representational State Transfer Application Programming Interface.REST APIs use HTTP requests to perform CRUD (Create, Read, Update, Delete) operations on resources, which are typically represented as URLs, and they follow a stateless client-server communication model.

Discuss the scope of var, let, and const

var: Function-scoped. Hoisted to the top of their containing function or global scope. Can be redeclared within the same scope. Not block-scoped.
let: Block-scoped. Not hoisted to the top of the block, which prevents "temporal dead zone" errors. Cannot be redeclared within the same scope. Can be reassigned.
const: Block-scoped. Not hoisted to the top of the block. Cannot be redeclared or reassigned within the same scope. Should be used for variables that should not change their value after initialization.

Tell us the use cases of null and undefined ?

null:

Used to explicitly represent the absence of any object value. Often set intentionally by developers to indicate that a variable or object property has no meaningful value or reference.

undefined:

Represents the absence of a defined value, typically for uninitialized variables or object properties. Automatically assigned to variables that have been declared but not assigned a value. Used by JavaScript as the default return value of a function that doesn't explicitly return anything.
